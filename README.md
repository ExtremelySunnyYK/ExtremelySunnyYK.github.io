# My portfolio and blog website

You can find it at [chiayong.me](ExtremelySunnyYK.github.io)

Built using Jekyll

## Dependencies

- [Gulp](https://gulpjs.com/)

## Getting Started

1.  `gem install jekyll sass bundler jekyll-minifier jekyll-sitemap`
1.  `npm install`
1.  `npm start`
